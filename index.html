Components of a Search Engine

Robot

A robot is a program that automatically traverses the hypertext structure of the Web by retreiving a web page document and following the hyperlinks on the page. In addition, some robots may access and store the title, meta tag descriptions, and some text on the page.

Database

The collection of information that is organized so that its contents can be accessed, managed, and updated. The web page that displays the results of your search has information from the database that is used by the search engine site. Some search engines use some of the content received form other search engines. Such as Yahoo! which gets its primary search results from Google.

Search Engine Optimization

Keywords

Spend time brainstorming terms and phrases that people may use when searching for your site. These terms are what describe your website or business.

Page Title

A descriptive page title which includes your company name or website name will help your site market itself. The page title is often displayed in the SERP and saved when a visitor bookmarks your page. Do not use the exact same title for each page of your website, include keywords in the page title that are appropiate for the page.

Heading Tags

Use structured tags, h1, h2, etc. to organize your content. If appropiate include some keywords in the text contained within the heading tags. Some search engines will give higher list position if keywords are used in the page title or heading. However, do not spam keywords, meaning repeating the words over and over within the content of the page. Search engines continue to become more sophisticated and spamming can prevent your page from being listed if it perceives you are trying to cheat the system.

Description

Write a few sentences about your website or business that makes it special and why people would want to visit. The description should be inviting and interesting so that someone would choose your site.

Description Meta Tag

A self-contained tag that is placed in the head section of a web page and is displayed on the SERP by some engines, such as Google.

Linking

Verify that all hyperlinks are working. Each page on your website should be reachable by a text hyperlink and should be descriptive. Do not use phrases such as "more info" or "click here." Should use keywords as appropiate.

Images and Multimedia

Search engine robots do not see the text within your images. Configure meaningful alt text for images and include relevant keywords.

Valid code

While it is not required by search engines that your code be validated, well structured and valid code is more easiliy processed by search engine robots which could help your placement in the search engine.

Content of Value

Your website should contain high-quality, well-organized content that is of your value to your visitors.

Additional Techniques for SEO

Map Your Site

An HTML site map is a web page with a map of the site that contains a hierarchical list of hyperlinks to the major pages of your website. This can be helpful for website visitors and assits search engine robots as they follow hyperlinks to your site. An XML site map is an XML file that is used by search engines only, not visitors. An XML site map provides the following information abouteach page to a search engine.

  • Date that each page was last modified
  • Indicator of how frequently each page changes
  • Priority level for each page

You can use an online sitemap generator to create a sitemap file.

Link Popularity

Search engines rate websites based on the number of sites that link to a particular website and the quality of those sites. Link popularity can determine its order in the SERP.

SMO

Make it easy for your current and potential website visitors by creating content of value that is shareable. Such as providing links to Twitter, Facebook, and Pinterest.
